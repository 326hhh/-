# PCA9685 机械爪驱动模块
基于 I2C 通信的 16 路 PWM 舵机驱动板，用于控制机械爪舵机。

## 硬件信息
- 驱动芯片：PCA9685
- 通信接口：I2C
- 默认地址：0x80
- 供电电压：舵机电源 5V，逻辑 3.3V
- 默认 PWM 频率：50Hz

## 接线说明
| PCA9685 引脚 | STM32 引脚 | 说明 |
| ---- | ---- | ---- |
| VCC | 5V | 电源输入 |
| GND | GND | 共地 |
| SDA | PB7 | I2C 数据线 |
| SCL | PB6 | I2C 时钟线 |

## 核心 API
- `PCA9685_Init()`：模块初始化，配置 PWM 频率与默认输出
- `PCA9685_SetAngle(uint8_t ch, float angle)`：设置指定通道舵机角度
- 通道范围：0 ~ 15，角度范围：0 ~ 180°

## 注意事项
- 机械爪最小闭合角度为 80°，低于此值可能导致舵机堵转
