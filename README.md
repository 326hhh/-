# 单片机通用驱动库
个人长期维护的单片机外设驱动、算法与通用组件库，基于 STM32 HAL 库开发，用于电赛、课程设计与个人项目快速复用。

├── docs/                                          # 芯片手册、开发笔记、原理图

## 移植使用方法
1.  复制目标模块的整个文件夹到你的工程目录中
2.  在工程里添加模块 `src` 目录下的 `.c` 文件
3.  将模块 `inc` 目录加入工程的头文件包含路径
4.  根据实际硬件修改头文件中的配置参数（引脚、地址、量程等）
5.  调用初始化函数后即可使用模块对外接口

## 基础规范
- 目录与文件名统一使用小写+下划线命名
- 头文件必须添加防重复包含宏
- 对外接口统一放在 `inc` 目录，内部私有函数与宏定义放在 `src` 内
- 每个模块配套独立 README，记录功能、接线、API 与注意事项
