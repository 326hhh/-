# PCA9685 机械爪驱动模块
基于 I2C 通信的 16 路 PWM 舵机驱动板，用于控制机械爪舵机。

## 硬件信息
- 驱动芯片：PCA9685
- 通信接口：I2C
- I2C 地址：7 位 0x40（写地址 0x80，读地址 0x81）
- 逻辑供电：VCC 接 3.3V（与 MCU 共电源）
- 舵机电源：V+ 接 5V **外接电源**
- 默认 PWM 频率：50Hz

## 接线说明
| PCA9685 引脚 | 连接 | 说明 |
| ---- | ---- | ---- |
| VCC | 3.3V | 芯片逻辑供电 |
| V+ | 5V 外接电源 | 舵机专用电源 |
| GND | GND | 共地 |
| SDA | PB7 | I2C 数据线 |
| SCL | PB6 | I2C 时钟线 |
| OE | GND | 输出使能，正常工作时必须接地(或者说低电平使能PCA9685工作) |

## 核心 API
- `PCA9685_Init()`：模块初始化，配置 PWM 频率与默认输出
- `Servo_SetAngle(uint8_t ch, uint8_t angle)`：设置指定通道舵机角度
- 通道范围：0 ~ 15，角度范围：0 ~ 180°

## 注意事项
- 机械爪最小闭合角度为 80°，低于此值可能导致舵机堵转

## 平台支持
- **已测试**：STM32F407ZGT6（ HAL 库）
